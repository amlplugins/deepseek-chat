# @amlplugins/deepseek-chat

Native `@ai-sdk/deepseek` SDK re-exported under the `@amlplugins` namespace.

> DeepSeek Chat — deepseek-chat / deepseek-coder completions via @ai-sdk/deepseek with streaming and tool use.

## Install

```bash
npm install @amlplugins/deepseek-chat
```

## Use

```ts
import { /* named exports from @ai-sdk/deepseek */ } from "@amlplugins/deepseek-chat";
// or
import sdk from "@amlplugins/deepseek-chat";
```

This package additionally re-exports `ai` under the `secondary` namespace.

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
